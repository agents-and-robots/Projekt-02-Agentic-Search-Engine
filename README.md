# Projekt-02-Agentic-Search-Engine
Eine lokale, CPU‑freundliche, offline‑fähige „Agentic Search Engine“, die:      Dokumente durchsucht      Fragen beantwortet      Suchstrategien plant      Ergebnisse bewertet      alles ohne Internet, ohne GPU, ohne große Modelle  Sie simuliert eine agentische Suche — ohne echte Websuche, damit sie auf jedem Laptop läuft.
